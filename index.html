NodeSchool Bristol

Join us on the 21st of February 2015 to learn node.js, an open source, cross-platform runtime environment for server-side and networking applications. Node is part of the MEAN stack; a great way to get into full-stack JavaScript development.

Reserve your space!

Learn to Code

Join us and learn to code with us. You'll learn the basics of node.js. It's a plus if you know some JavaScript beforehand, but if not we can help you out.

Self-guided

You'll go through a set of self-paced challenges, with no boring talks. All challenges are open source and available online on nodeschool.io. This means you can continue learning after the event.

For Everyone

We think coding is for everyone, and everyone should feel welcome at our event. This means that we are expecting all participants to be lovely to each other and respect the code of conduct.
